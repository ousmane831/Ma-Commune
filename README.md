Plateforme Web de la Commune de Niakhar
________________________________________
1. Contexte et justification
Dans le cadre de la modernisation de la communication locale, la commune de Niakhar souhaite mettre en place une plateforme web responsive permettant de diffuser des informations officielles et de favoriser l’interaction avec les citoyens.
Actuellement, la communication entre la commune et les citoyens est limitée, peu centralisée et manque d’interactivité. Cette plateforme vise à combler ce manque en offrant un espace numérique accessible à tous.
________________________________________
2. Objectifs du projet
2.1 Objectif général
Mettre en place une plateforme web moderne permettant :
•	La diffusion d’informations officielles de la commune
•	L’interaction entre citoyens et autorités locales
2.2 Objectifs spécifiques
•	Permettre aux citoyens de consulter les publications
•	Permettre aux citoyens d’interagir (likes, commentaires)
•	Offrir un système de mention entre utilisateurs
•	Permettre le partage des publications sur les réseaux sociaux
•	Fournir un espace d’administration pour gérer le contenu
________________________________________
3. Description générale du système
La plateforme sera accessible via un navigateur web (ordinateur, tablette, smartphone) et sera responsive.
Elle comportera deux types d’utilisateurs :
•	Citoyen
•	Administrateur
________________________________________
4. Fonctionnalités
4.1 Fonctionnalités pour les citoyens
•	Consulter les publications de la commune
•	Liker une publication
•	Commenter une publication
•	Mentionner d’autres utilisateurs (@nom)
•	Recevoir des notifications (mention, réponse…)
•	Partager une publication via :
o	WhatsApp
o	Facebook
o	Autres réseaux sociaux
________________________________________
4.2 Fonctionnalités pour l’administrateur
•	Publier des annonces (texte, image)
•	Modifier ou supprimer une publication
•	Gérer les commentaires (modération)
•	Gérer les utilisateurs (activer, désactiver)
•	Suivre l’activité de la plateforme
________________________________________
5. Contraintes techniques
•	Plateforme web responsive (mobile + desktop)
•	Développement backend : Django
•	Base de données : PostgreSQL ou SQLite
•	Frontend : HTML, CSS, Bootstrap ou JavaScript
•	Sécurité : authentification obligatoire pour les interactions
•	Hébergement web requis

Ce projet vise à digitaliser la communication de la commune de Niakhar en mettant en place une plateforme moderne, interactive et accessible. Il permettra d’améliorer la transparence, la participation citoyenne et la diffusion rapide des informations locales.
